# 🛒 Quantium Retail Analytics Project

This project was completed as part of the Quantium Data Analytics Virtual Internship.

## 📊 Project Overview

The goal of this project was to analyse retail transaction data and evaluate the performance of a store trial conducted in three stores (77, 86, 88).

The analysis was divided into two main parts:
- Exploratory Data Analysis (EDA)
- Trial Store Performance Evaluation

---

## 🔍 Key Tasks

### 1. Data Exploration
- Cleaned and processed transaction data
- Analysed customer segments and purchasing behaviour

### 2. Trial Analysis
- Compared trial stores with control stores
- Used correlation to identify similar stores
- Measured performance using:
  - Total Sales
  - Number of Customers
  - Transactions per Customer

---

## 📈 Key Insights

- All trial stores showed an increase in sales
- Growth was primarily driven by increased customer numbers
- Stores 86 and 88 showed the strongest performance improvements

---

## 💡 Recommendations

- Roll out the trial strategy to similar stores
- Focus on increasing customer traffic
- Enhance promotions to improve purchase frequency
